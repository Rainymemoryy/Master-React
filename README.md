# Shopee

## Deploy https://shopee-seven.vercel.app/

## Mô tả ứng dụng

Trang web sử dụng RectJS với các framework và thư viện như:

1. Redux, Redux Toolkit
2. Styled component
3. React router dom
4. React hook form
